## Description
簡単なタスクアプリです.  

## Requirement
- expo-cli
- typescript

## Usage
1. clone
```bash
$ git clone https://github.com/FukeKazki/ReactNative-ToDo-App-TS.git
```
2. run
```bash
$ expo install
$ expo start
```
